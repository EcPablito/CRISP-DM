# Aplicación CRISP-DM Ponte Selva

Aplicación Streamlit para análisis CRISP-DM con modelos SARIMAX y Prophet.

## Ejecutar localmente

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Archivos

- app.py
- requirements.txt
- dataset_simulado_retroalimentacion_calidad_ventas_canales_codificados.csv
